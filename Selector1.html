Selector in HTML

i. Element Selector

Element selector selects the HTML elements based on the element name.

ii. ID Selector

Id selector uses the id attribute of an HTML element to select a specific element. The ID of an element is unique within a page.Hence the ID selecotor is used to select only one unique element using the #followed by ID name.

iii. Class Selector

The class selector selects HTML elements with a specific value of the class attribute. It uses the .class_name{} for accessing the class. Also class is not unique that means mulitple elements can use the same class name.
NOTE: A html element can have more than one class.
