HTML(HYPER TEXT MARKUP LANGUAGE)

Html which stands for the hypertext markup language is the markup language that is used in web programming to build the solid structure of the web doucments or web apps. Basically the term hypertext referes to that kind of text which can be further studied detailly in non sequential order as we want & markup means here the tag is used for marking up the document so that it helps us in the navigation and formats the code in well structured way.
